"Problem:1 You are given a string (STR) of length N, consisting of only the lower case English alphabet. Your task is to remove all the duplicate occurrences of characters in the string. Input: abcadeecfb

OUTPUT: abcdef

        let str1 = 'abcadeecfb';
        let s = new Set();

        for(let i = 0; i < str1.length; i++){

            s.add(str1[i]);    

        } 
        let result = '';

        for(let x of s) {
            result = result + x;
        }
        console.log(result);
